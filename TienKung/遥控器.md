###  开机
先拉开急停按钮，背部电池下方左侧大开机键**短按**，右侧开机键**短按长按**开机，遥控器**短按长按**。开机后自检查等音效。

随后按**A键**实现遥控器控制，等待音效。

随后按**D键**进入"准备姿势"

将机器人放下接触地面后注意前后左右是否有偏移。

随后**长按A键**进入**站立**

### 模式
```mermaid
graph LR
	A([准备姿势]) -- 长按A --> B[站立]
    B -- H左拨后回正 --> C[走路]
    D[跑步] -- 左拨后回正 -->  C
    C -- H右拨后回正 --> D
    C --长按A--> B
```
H键在遥控器右上方
